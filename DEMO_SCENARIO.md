# Demo Scenario

**가게 문구 미니 생성기 (Store Copy Mini Generator)**

> 단 6개의 입력값으로 인스타그램 & 네이버 플레이스용 홍보 문구를 즉시 생성하는 프로토타입

---

## 이 데모가 보여주는 문제

### 소상공인, 특히 동네 가게 사장님들은 마케팅 카피 작성에 어려움을 겪습니다.

- **시간 부족**: 가게 운영 + 매장 관리로 바빠 홍보 문구를 고민할 여유가 없음
- **전문성 부족**: "우리 가게를 어떻게 한 줄로 소개하지?" — 카피라이팅 경험 없음
- **플랫폼 피로**: 인스타그램 프로필, 네이버 플레이스 소개글, 리뷰 답글 — 써야 할 곳은 많은데 매번 새로 쓰기엔 부담
- **돌려막기 현상**: 다른 가게 문구를 베끼거나, 몇 달째 같은 문구를 쓰는 악순환

**이 데모가 해결하는 방식**:
> 가게 정보를 6개 항목(가게명, 업종, 지역, 장점, 타겟 고객, 연락처)만 입력하면, 4가지 유형의 홍보 문구를 한 번에 생성합니다. 복사 버튼 하나로 클립보드에 바로 복사 가능합니다.

---

## 30초 데모 흐름

### Open → Fill → Generate → Copy

```
1. [  0초] index.html을 브라우저에서 연다
2. [  5초] 우측 하단의 "샘플 가게 불러오기" 버튼 클릭
3. [ 10초] "모모네일" 샘플 클릭 → 6개 입력칸 자동 완성
4. [ 15초] "문구 생성하기" 버튼 클릭
5. [ 20초] 생성된 문구 확인 (4개 영역 × 3~5개 문구)
6. [ 25초] 원하는 문구 옆 "복사" 버튼 클릭 → 클립보드 복사 완료
7. [ 30초] 인스타그램 or 네이버 플레이스에 붙여넣기
```

> **결과**: 30초 안에 샘플 가게의 홍보 문구를 생성하고 복사까지 완료할 수 있습니다.

## 3분 데모 흐름

3분 데모는 **실제 사용자가 자신의 가게 정보를 직접 입력하는 시나리오**로 진행합니다.

### 1단계: 소개 및 문제 공감 (0:00 ~ 0:40)

- "소상공인 사장님들이 마케팅 문구 쓰느라 고생한다는 얘기를 들어보셨죠?"
- 실제 사례: 동네 카페 사장님이 인스타 프로필 문구 때문에 스트레스 받는 상황
- "이 도구는 그 문제를 6개 입력값으로 해결합니다."

### 2단계: 샘플로 빠르게 체험 (0:40 ~ 1:10)

- "샘플 가게 불러오기" → "소소카페" 선택
- 가게명/업종/지역/장점/타겟/연락처가 자동 입력되는 모습 확인
- "문구 생성하기" 클릭 → 4개 영역 결과 확인
- "복사" 버튼 클릭 → "✓" 표시로 복사 완료 확인

### 3단계: 직접 입력 데모 (1:10 ~ 2:00)

- "이번엔 제가 직접 가게 정보를 입력해볼게요"
- 초기화 버튼 클릭 → 빈 폼 확인
- 직접 입력 (예: "달콤베이커리", "베이커리", "망원동",
  "수제 생크림, 주말 브런치, 포장 가능", "주말에 가족과 방문하는 30~40대", "인스타 DM 주문")
- "문구 생성하기" → 결과 확인

### 4단계: 결과 활용법 설명 (2:00 ~ 2:30)

- "한 줄 소개문은 인스타그램 프로필 소개글에"
- "인스타그램 프로필 문구는 네이버 플레이스 소개글에"
- "리뷰 답글 템플릿은 고객 리뷰에 맞춰서"
- 실제 화면에 붙여넣기 시연 (브라우저 탭 전환 or 이미지)

### 5단계: 데모 한계와 향후 방향 (2:30 ~ 3:00)

- "이 데모는 자동화되어 있지 않습니다 — 직접 확인하고 복사하는 구조"
- "데이터는 전부 가상이며, 실제 인스타그램/네이버와 연동되지 않습니다"
- "확장 가능성: 업종별 템플릿 고도화, AI 문장 생성 연동, 플랫폼 자동 포스팅"

## 사용자가 클릭해볼 순서 (Step by Step)

### 기본 시나리오: 가게 정보 입력 → 문구 생성 → 복사

#### Step 1: 페이지 열기
- `index.html`을 브라우저에서 더블클릭 또는 로컬 서버로 오픈
- **확인 포인트**: "✏️ 가게 문구 미니 생성기" 제목 표시, 6개 입력 폼, 3개 버튼 확인

#### Step 2: 샘플 가게 불러오기
- [샘플 가게 불러오기] 버튼 클릭
- **확인 포인트**: 6개 샘플 가게 버튼이 2~3열 그리드로 표시됨

#### Step 3: 샘플 선택
- 원하는 샘플 클릭 (예: "모모네일 — 네일샵")
- **확인 포인트**: 입력 폼 6개 항목이 자동으로 채워짐
- 샘플 섹션이 자동으로 닫힘

#### Step 4: 문구 생성
- [문구 생성하기] 버튼 클릭
- **확인 포인트**: 4개 결과 영역 표시
  - ✅ 한 줄 소개문 (5개)
  - ✅ 인스타그램 프로필 문구 (3개)
  - ✅ 예약/문의 CTA (3개)
  - ✅ 리뷰 답글 템플릿 (3개)

#### Step 5: 문구 확인
- 각 문구를 읽어보고 마음에 드는 선택
- 줄바꿈이 포함된 문구(프로필 문구 등)는 `<br>`로 렌더링 확인
- **확인 포인트**: 다크/라이트 모드에서 텍스트 가독성

#### Step 6: 문구 복사
- 원하는 문구 옆 [복사] 버튼 클릭
- **확인 포인트**: 버튼이 "✓"로 바뀌며 1.5초 후 원래대로 복귀
- 클립보드에 정상 복사되었는지 확인 (텍스트 에디터 등에 붙여넣기)

#### Step 7: 초기화
- [초기화] 버튼 클릭
- **확인 포인트**: 모든 입력칸 빈칸, 결과 영역 숨김, 토스트 메시지 표시

### 심화 시나리오: 다른 가게로 전환

#### Step 8: 다른 샘플 적용
- 다시 [샘플 가게 불러오기] → 다른 가게 선택 (예: "리얼피부")
- [문구 생성하기] → 새로운 결과 확인
- **확인 포인트**: 입력값이 완전히 교체되고 결과도 새로 생성됨

#### Step 9: 직접 입력
- [초기화] → 모든 항목을 직접 입력 (위 3분 데모 3단계 참고)
- [문구 생성하기]
- **확인 포인트**: 빈칸이 있을 경우 토스트 경고 ("필수 입력값을 모두 채워 주세요.")

#### Step 10 (선택): 여러 문구 비교
- 생성된 문구 중 마음에 드는 것을 여러 개 복사해서 메모장에 붙여넣기
- **확인 포인트**: 줄바꿈, 특수문자가 정상 보존됨

## 실제 사용 방식 (Copy & Paste to Instagram/Naver)

이 도구는 **자동 포스팅을 하지 않습니다.** 사람이 직접 검토하고 선택해서 붙여넣는 방식입니다.

### 인스타그램 프로필

```
1. 생성된 "인스타그램 프로필 문구" 중 하나 선택
2. [복사] 버튼 클릭
3. 인스타그램 앱 → 프로필 편집 → 소개글에 붙여넣기
4. 저장

예시 복사 내용:
📍 성수
🏠 모모네일 (네일샵)
✨ 손상 적은 케어 · 차분한 분위기
💬 과하지 않은 깔끔한 네일을 원하는 직장인 환영
📩 인스타 DM 예약
```

### 네이버 플레이스 소개글

```
1. 생성된 "한 줄 소개문" 중 마음에 드는 것 선택
2. [복사] 버튼 클릭
3. 네이버 플레이스 → 내 정보 → 가게 소개글 수정 → 붙여넣기
4. 저장

예시 복사 내용:
성수에서 손상 적은 케어을(를) 제공하는 네일샵, 모모네일입니다.
차분한 분위기과(와) 합리적인 가격(으)로 과하지 않은 깔끔한 네일을 원하는 직장인을(를) 위한 최상의 경험을 선사합니다.
```

### 리뷰 답글

```
1. 고객 리뷰의 톤에 맞는 템플릿을 "리뷰 답글 템플릿"에서 선택
2. [복사] → 리뷰 관리 페이지에서 붙여넣기
3. 필요 시 문장 앞에 고객 닉네임 추가 등 미세 조정

예시 복사 내용:
모모네일을(를) 방문해 주셔서 감사합니다. 손상 적은 케어(으)로 다음에도 편안하게 관리받으실 수 있도록 준비하겠습니다.
```

### 예약/문의 CTA

```
1. 생성된 CTA 문구 복사
2. 인스타그램 하이라이트, 네이버 예약 버튼 옆 설명, 또는 카카오톡 채널 소개글에 활용

예시 복사 내용:
인스타 DM 예약(으)로 편하게 문의해 주세요. 모모네일이(가) 손상 적은 케어(으)로 맞이하겠습니다.
```

## 자동화하지 않는 부분

이 데모는 **일부러 자동화하지 않은 영역**이 있습니다. 이는 실제 소상공인 사용 환경에서의 신뢰성과 통제권을 보장하기 위함입니다.

| 자동화하지 않는 부분 | 이유 |
|---|---|
| **인스타그램/네이버 자동 로그인** | 보안 문제. 사용자의 SNS 계정 정보를 다루지 않음 |
| **자동 포스팅 / DM 발송** | 사용자가 직접 검토하고 승인해야 부적절한 문구 발송 방지 |
| **실제 가게 데이터 크롤링** | 허가 없는 데이터 수집은 법적/윤리적 문제 발생 가능 |
| **이미지/사진 생성** | 본 도구는 텍스트 카피에 집중 (이미지는 별도 도구 활용) |
| **문구 자동 선택** | 생성된 여러 옵션 중 사용자가 직접 고르는 것이 더 좋은 결과 |
| **외부 API 호출 (AI 등)** | 오프라인에서도 동작 가능한 독립형 HTML 데모로 설계 |
| **사용자 데이터 저장** | 개인정보 보호를 위해 모든 데이터는 브라우저 메모리에서만 처리 |

### 핵심 원칙
> **"사람이 결정하고, 도구가 도와준다"**
> 
> 이 도구는 소상공인의 카피라이팅을 **완전히 대체**하지 않고, **초안을 빠르게 제공**하여
> 사장님이 최종 검토 후 직접 사용할 수 있게 하는 것을 목표로 합니다.

## 리뷰어가 확인할 장면

데모 리뷰 시 다음 장면들을 중점적으로 확인해 주세요.

### 1. 입력 폼 — 6개 필드의 완성도
- [ ] 모든 입력칸에 placeholder가 적절한 예시를 보여주는가
- [ ] 2열 그리드 레이아웃이 다양한 화면 크기에서 깨지지 않는가
- [ ] 입력값 유효성 검사가 동작하는가 (빈칸 → 토스트 경고)
- [ ] 키보드 탭 이동이 자연스러운가

### 2. 샘플 가게 불러오기
- [ ] 버튼 클릭 시 샘플 섹션이 부드럽게 나타나는가
- [ ] 6개 샘플이 모두 정상 표시되는가
- [ ] 샘플 선택 → 폼 채움 → 샘플 섹션 자동 닫힘 — 이 흐름이 끊김 없는가
- [ ] 연속으로 다른 샘플을 선택했을 때 폼이 교체되는가

### 3. 문구 생성 결과
- [ ] 4개 영역이 모두 올바르게 표시되는가 (한 줄 소개문 5개, 프로필 3개, CTA 3개, 리뷰 답글 3개)
- [ ] 각 문구가 입력값(가게명, 업종, 장점 등)을 정확히 반영하는가
- [ ] 줄바꿈(`\n`)이 포함된 문구(프로필 문구)가 화면에서 정상 렌더링되는가
- [ ] 생성 후 결과 영역으로 자동 스크롤되는가

### 4. 복사 기능
- [ ] [복사] 버튼 클릭 시 "✓"로 변경되는가
- [ ] 1.5초 후 원래 "복사" 텍스트로 복귀하는가
- [ ] 클립보드에 정상 복사되었는가 (외부 앱에 붙여넣기 테스트)
- [ ] 줄바꿈이 포함된 텍스트도 클립보드에 정상 보존되는가
- [ ] 여러 번 연속 복사 시 버그가 없는가

### 5. 반응형 디자인
- [ ] 데스크톱(768px+)에서 2열 입력 폼이 깔끔한가
- [ ] 모바일(600px↓)에서 1열로 전환되는가
- [ ] 모바일에서 샘플 버튼 그리드가 2열 → 1열로 줄어드는가
- [ ] 버튼 영역이 화면 폭에 따라 래핑되는가

### 6. 사용자 경험
- [ ] 토스트 메시지가 적절한 위치와 시간으로 표시되는가
- [ ] 공지 섹션(⚠️)의 주의 문구가 눈에 띄는가
- [ ] "다음 단계" 섹션이 데모 목적을 잘 설명하는가
- [ ] 전체적으로 가독성 있는 폰트 크기와 간격인가

### 7. 에지 케이스
- [ ] 모든 입력칸을 비우고 생성 시도 → 경고 메시지 표시
- [ ] 장점을 1개만 입력했을 때 → 나머지 슬롯이 기본값으로 대체
- [ ] 매우 긴 가게명이나 장점 입력 → 레이아웃이 깨지지 않는가
- [ ] 특수문자(이모지, 따옴표 등) 포함 시 정상 동작하는가
