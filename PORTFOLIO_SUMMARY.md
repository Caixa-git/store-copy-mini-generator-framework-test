# Portfolio Summary

## 한 줄 소개

**스토어 카피 미니 생성기 프레임워크** — 규칙 기반 템플릿 엔진으로 네일샵, 카페 등 소상공인이 **1초 만에** 매장 홍보 문구를 생성할 수 있는 오프라인 툴킷입니다.

## 타겟 사용자

| 타겟 | 세부 설명 |
|------|-----------|
| **네일샵 원장** | 네일 아트 시안별 홍보 문구, 시즌 이벤트 문구, SNS 업로드용 카피가 필요하나 매번 작성을 부담스러워함 |
| **카페 사장님** | 신메뉴 안내, 시즌 음료 홍보, 오픈/이벤트 공지 등 매장 운영에 필요한 짧은 카피를 빠르게 원함 |
| **소상공인 일반** | 마케팅 전담 인력이 없는 1인/소규모 매장 운영자 — 간단한 입력만으로 즉시 사용 가능한 문구 필요 |

## 해결하는 문제

소상공인은 매장 홍보가 필요하지만:
- **시간 부족**: 하루 12시간 매장 운영 중 카피 작성에 쓸 시간이 없음
- **비용 부담**: 마케터 고용 또는 AI 서비스 구독은 월 5~50만 원 부담
- **번거로움**: AI 서비스는 가입, 로그인, 프롬프트 입력 등 여러 단계를 거쳐야 함
- **일관성 부족**: 매번 작성하는 문구 스타일이 달라 브랜드 정체성이 흐려짐

→ **프레임워크는 이 모든 과정을 단순한 UI + 규칙 기반 템플릿으로 해결**하여 누구나 1클릭으로 완성된 문구를 얻을 수 있게 합니다.

## 주요 기능 5개

| # | 기능 | 설명 |
|---|------|------|
| 1 | **템플릿 기반 카피 생성** | 업종별(네일샵, 카페 등) 사전 정의된 템플릿에 변수만 채워 즉시 완성 문구 출력 |
| 2 | **카테고리 분류 자동화** | 업종 > 세부 카테고리 > 템플릿 계층 구조로 원하는 문구 유형을 빠르게 탐색 |
| 3 | **프레임워크 검증 (Validation)** | 필수 변수 검사, 길이 제한 검증, 템플릿 구문 오류 체크로 깨진 출력 방지 |
| 4 | **1-클릭 복사/내보내기** | 생성된 문구를 클립보드 복사, SNS 플랫폼 별 맞춤 길이 조정 (인스타/네이버/카카오) |
| 5 | **확장 가능한 플러그인 구조** | 새로운 업종/템플릿을 JSON 또는 YAML 정의만으로 추가 가능 |

## Before / After

### Before: 카피 작성에 어려움을 겪는 상황

```
😫 "새 네일 시안 나왔는데 인스타에 뭐라고 올리지?"
   → 30분 고민, 폰으로 두 줄 적다가 포기
   → 결국 "새 시안 나왔어요 ❤️" 로 때움

😫 "겨울 시즌 메뉴 홍보해야 하는데..."
   → 카피 작성 + 이미지 편집 + 업로드 = 1시간 소요
   → 일주일 뒤에도 공지 못 올린 상태

😫 "블로그에도 홍보글 쓰래..."
   → AI 써보려고 했는데 회원가입하고 프롬프트 쓰는 것도 일
   → 결국 '다음에 하지 뭐' 반복
```

### After: 1-클릭 생성 후 자유로운 시간

```
😊 새 시안 인스타 업로드
   → 업종(네일샵) → 카테고리(시안소개) 선택 → 변수(시안명, 가격) 입력
   → Generate 클릭 → 완성된 문구 복사 → 붙여넣기
   👉 **총 소요 시간: 10초**

😊 시즌 메뉴 SNS 3종 업로드
   → 동일한 템플릿으로 인스타/블로그/카톡 채널 문구 각각 생성
   → 각 플랫폼에 맞는 길이로 자동 조정
   👉 **총 소요 시간: 30초**

😊 일관된 브랜드 톤 유지
   → 동일한 템플릿 세트로 모든 문구가 같은 스타일 유지
   → 매장의 브랜드 정체성을 해치지 않음
   👉 **매일 꾸준한 홍보 가능**
```

## 포트폴리오에서 강조할 점

### ✅ No API Dependency (API 의존성 없음)

- 순수 **규칙 기반 템플릿 엔진**으로 동작 — 외부 API 키, 구독, 네트워크 연결 불필요
- 오프라인 환경에서도 완벽히 작동 — 매장 내 PC, 태블릿에서 즉시 사용 가능
- AI 서비스 중단/요금 인상 등의 리스크에서 완전히 자유로움
- 사용자 데이터가 외부로 전송되지 않아 개인정보 보호에도 유리

### ✅ Rule-Based (룰 기반 템플릿 시스템)

- **정해진 규칙대로 동작**하여 예측 불가능한 출력이 발생하지 않음
- AI의 할루시네이션(Hallucination) 문제가 원천 차단됨
- 모든 템플릿은 사전 검증되어 깨진 문법, 부적절한 표현이 나올 수 없음
- 사용자가 템플릿을 직접 작성/수정 가능 — **프로그래밍 지식 불필요**

### ✅ Framework Validation (프레임워크 검증)

- **입력 검증**: 필수 변수 누락 시 즉시 에러 피드백 → 사용자 혼란 방지
- **템플릿 구문 검증**: 잘못된 템플릿 문법 감지 → 사전 차단
- **출력 길이 검증**: 플랫폼별 제한 글자 수 초과 시 자동 조정 또는 경고
- **일관성 검증**: 동일 업종 내 템플릿 스타일이 일관되게 유지되는지 체크
- 검증 실패 시 **명확한 에러 메시지**와 **해결 방법**을 함께 제시
