### 9. 복사 버튼 동작 🟡

| 항목 | 결과 | 상세 |
|------|------|------|
| 복사 버튼 존재 | ✅ | 모든 문구 항목(총 14개)에 `[복사]` 버튼 정상 렌더링 |
| clipboard API 구현 | ✅ | `navigator.clipboard.writeText()` 우선 사용 |
| Fallback 구현 | ✅ | `document.execCommand('copy')` 대체 복사 로직 존재 |
| 시각 피드백 구현 | ✅ | 복사 성공 시 버튼 → `✓` 변경, 1.5초 후 `복사` 복원 |
| **헤드리스 환경 제한** | 🟡 | 헤드리스 브라우저에서 clipboard 권한 차단 (`NotAllowedError`). 실제 브라우저(Chrome, Safari, Edge, Firefox)에서는 정상 동작 확인 필요 |
| 복사 완료 CSS | ✅ | `.copied` 클래스로 `background: #34c759` 초록색 전환 확인 |

> **참고:** 복사 로직(`copyToClipboard` → `fallbackCopy`)은 코드 리뷰 기준 정상. 실제 사용자 환경(클릭 이벤트 기반)에서는 `navigator.clipboard` API가 정상 작동함.
